3.5.1 搜索算法

  1. find()用于搜索无序集合中是否存在某值。范围由iterator[first,last)标出。返回iterator指向该值/last。
  2. binary_search()用于有序集合的搜索。返回true/false。
  3. count()返回数值相符的元素数目。
  4. search()比对某个容器内是否存在某个子序列。返回iterator指向子序列起始处/last。
