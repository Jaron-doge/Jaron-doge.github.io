第一章 C++编程基础

1.1 如何撰写C++程序

  • class的定义分为两部分：

    • 头文件(header file)：声明该class所提供的各种操作行为(operation)
    • 程序代码文件(program text)：操作行为的实现内容(implementation)

  • 语句是C++程序的最小独立单元

  • using namespace std;
<!--￼0-->

  • 命名空间：一种将库名封装起来的方法,通过这种方法，避免和应用程序发生命名冲突的问题。

    • 命名冲突：在应用程序内两个不同的实体(entity)具有相同名称，导致程序无法区分两者。命名冲突发生时，程序必须等到该命名冲突获得解析(resolve)后，才能继续执行。

1.3 撰写表达式

  • ++C与C++

    • ++C:原值先递增，之后才被拿来使用
    • C++:原值先供给表达式进行运算，然后才递增

  • 短路求值

    ||:第一个表达式为true，后续表达式不再求值

    &&:第一个表达式为false，后续表达式不再求值
